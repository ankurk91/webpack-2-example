# [Webpack](https://github.com/webpack/webpack) by example

> Lets learn WebPack : The module bundler

### Prerequisites
* node js ^6.7.1
* npm ^3.10.8
* webpack ~1.13.3

### Quick Start
* Clone this repo
```
# Install dependencies
npm install

# Run dev server at localhost:9000
npm run dev

```
* Generate dist (production ready) files
```
npm run build
```
